# RhythmEcho Beta 

A delay plugin that follows *your* playing instead of locking to a project BPM.

RhythmEcho is technically a VST3 delay plugin, but so much more. Feed it any audio — guitar, drums, piano, a synth stab, even pads — and it creates echoes timed to *your* rhythm. Speed up, slow down, go loose or locked tight: the echoes follow. No tempo map, no grid, no sync, no BPM unless you want it. Just you and the time you're making.

The idea came from listening to J-Dilla loops and Scratch Perry dub mixes where the echo isn't on the click, it's on the feel. RhythmEcho delays are *alive* — they breathe with the track instead of sitting on top of it. 

See the [Wiki](../../wiki/) for how to use RhythmEcho.
